## 📋 核心技术命令

### 4 个核心命令（必须掌握）

```bash
# 1. 复制应用
sudo cp -R /Applications/原应用.app /Applications/应用分身.app

# 2. 修改 Bundle ID
sudo /usr/libexec/PlistBuddy -c "Set :CFBundleIdentifier com.example.app2" \
  /Applications/应用分身.app/Contents/Info.plist

# 3. 重新签名
sudo codesign --force --deep --sign - /Applications/应用分身.app

# 4. 启动多开
open -n /Applications/应用分身.app
```

**详细说明**：请查看 [docs/技术原理说明.md](docs/技术原理说明.md#核心步骤详解)
