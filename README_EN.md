# WeChatDouble

> 🔧 **Mac App Multi-Instance Tool Technical Research - Using WeChat as Example**

---

## 📖 Project Introduction

This is a **pure technical research project** that explores the implementation principles of app multi-instance, signature modification, and icon customization on the macOS platform.

**Important Note**: This project uses WeChat as a **demonstration example**, but the technical principles and methods shown are applicable to the vast majority of applications on the macOS platform.

This project provides detailed instructions on how to achieve the following on personal devices:

- ✅ **Dual/Multi-Instance**: Run multiple instances of the same application on one device
- ✅ **Modify App Signature**: Customize application signature configuration
- ✅ **Change App Icon**: Personalize application icons
- ✅ **Detailed Principles**: Complete technical principle explanations and implementation steps

**This project is completely open source and provided for research and learning purposes.**
