# MacAppMultiLauncher

> 🔧 **Mac 软件多开工具技术研究 - 通用应用分身方案**

---

## 📖 项目简介

本项目是一个**纯技术研究项目**，探索 macOS 平台上应用多开、签名修改、图标定制等技术的实现原理。

**重要说明**：本项目以微信应用作为**示范样例**，但所展示的技术原理和方法适用于 macOS 平台上的绝大多数应用程序。

本项目详细介绍如何在个人设备上实现应用的：

- ✅ **双开/多开**：在同一设备上运行多个应用实例
- ✅ **修改应用签名**：自定义应用签名配置
- ✅ **更换应用图标**：个性化定制应用图标
- ✅ **原理详解**：完整的技术原理说明和实现步骤

**本项目完全开源，供给需要的人进行研究进行学习使用。**
