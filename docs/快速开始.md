# 快速开始 - 5 分钟搞定微信双开

> 最简单的一键双开方案，无需复杂操作

---

## 🚀 一键双开（推荐）

### 步骤 1：下载项目

克隆或下载本项目到本地：

```bash
git clone <项目地址>
cd WeChatDouble
```

### 步骤 2：执行脚本

```bash
sudo bash wechat-dual.sh
```

**就这么简单！** 脚本会自动完成所有配置并启动双开微信。

---

## 📝 手动操作（了解原理）

如果想了解每个步骤的原理，可以手动执行以下 4 个核心命令：

### 1️⃣ 复制应用

```bash
sudo cp -R /Applications/WeChat.app /Applications/微信双开.app
```

**作用**：创建微信的完整副本

### 2️⃣ 修改 Bundle ID

```bash
sudo /usr/libexec/PlistBuddy -c "Set :CFBundleIdentifier com.tencent.xinWeChat2" \
  /Applications/微信双开.app/Contents/Info.plist
```

**作用**：让系统识别为独立应用

### 3️⃣ 重新签名

```bash
sudo codesign --force --deep --sign - /Applications/微信双开.app
```

**作用**：绕过系统签名验证

### 4️⃣ 启动双开

```bash
open -n /Applications/微信双开.app
```

**作用**：强制打开新实例

## ❓ 常见问题

### Q: 提示"应用已损坏"怎么办？

重新签名即可：

```bash
sudo codesign --force --deep --sign - /Applications/微信双开.app
```

### Q: 图标带斜杠但能启动？

**这是正常现象！** 系统标记的"可疑应用"提示，不影响使用。

**如需修复**，运行一键修复脚本：

```bash
bash fix-icon-slash.sh
```

或手动修复：

```bash
# 1. 清除隔离属性
xattr -cr /Applications/微信双开.app

# 2. 修复权限
sudo chown -R $USER:staff /Applications/微信双开.app
sudo chmod -R 755 /Applications/微信双开.app

# 3. 重新签名
codesign --force -s - --timestamp=none /Applications/微信双开.app
```

详细说明请查看：[技术原理说明](技术原理说明.md#q15-图标带斜杠但能启动重要)

### Q: 数据会混淆吗？

**不会！** 两个微信的 Bundle ID 不同，数据完全隔离。

### Q: 微信更新后怎么办？

删除双开应用，重新运行脚本即可。
